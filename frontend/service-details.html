📖 Usage Guide

Protected routes require an X-API-Key header. Requests without a valid key will receive a 401 Unauthorized response.

# ❌ Without key — returns 401
curl http://localhost:30080/SERVICE/endpoint

# ✅ With key — request goes through
curl -H "X-API-Key: your-api-key" \
http://localhost:30080/SERVICE/endpoint

# Python requests
requests.get(
"http://localhost:30080/SERVICE/endpoint",
headers={"X-API-Key": "your-api-key"})

# JavaScript fetch
fetch("http://localhost:30080/SERVICE/endpoint", {
headers: { "X-API-Key": "your-api-key" }
})
